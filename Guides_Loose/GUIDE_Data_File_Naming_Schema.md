# GUIDE: Data File Naming Schema

## 1. PRIMARY SCHEMAS

- **INS_InstructionsTitle:** Main system instructions and directives.
- **KD_Blueprints_[Name]:** Data sets of environments and worlds.
- **INS_Guide_[Name]:** Subsets of instructions to be executed after main instructions.
- **P_Skill_[Name]:** Professional Base Personas used for basic functions.

## 2. ATMOSPHERIC LOGIC SCHEMA (SECTORS & LOCATIONS)

Format: **[Chemical/Elemental Descriptor] + [Visceral Condition/Atmosphere]**
- *Examples:* The Mercury Weep, The Cobalt Hemorrhage.

## 3. TACTICAL ACADEMIC SCHEMA (NODES & ENTITIES)

Format: **[Function] + [Sector Code]** or **[Anomaly Type] + [Void Status]**
- *Examples:* Eraser-Exon-9, Mimic-Chameleon-Null.
